Una {\bfseries{estructura}} es una colección de variables (llamadas {\bfseries{miembros}}) que pueden ser de diferentes tipos de datos, agrupadas bajo un nombre común en un bloque de memoria contiguo. Son fundamentales para organizar datos relacionados y crear tipos de datos complejos.

{\bfseries{Características clave\+:}}
\begin{DoxyItemize}
\item Agrupan datos heterogéneos bajo un mismo nombre
\item Ocupan memoria contigua para todos sus miembros
\item No incluyen métodos (funciones) como en otros lenguajes
\item También se conocen como \+\_\+\char`\"{}registros\char`\"{}\+\_\+ en algunos contextos

{\bfseries{Consideraciones de ámbito\+:}}
\begin{DoxyItemize}
\item Las estructuras definidas fuera de funciones tienen ámbito global
\item Las variables declaradas junto con la definición son globales
\item Para uso local, definir la estructura globalmente y declarar variables dentro de funciones
