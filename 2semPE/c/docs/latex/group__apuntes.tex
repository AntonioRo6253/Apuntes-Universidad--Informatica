\item if\+: la sintaxis general es\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{if} (condición) \{}
\DoxyCodeLine{    \textcolor{comment}{/* sentencia(s) */}}
\DoxyCodeLine{\}}

\end{DoxyCode}


Donde \char`\"{}condición\char`\"{} es una expresión booleana o integral; en C se considera verdadera si el resultado es distinto de 0 y falsa si es 0. Desde C99 puede usarse el tipo \+\_\+\+Bool o el alias bool (incluyendo \texorpdfstring{$<$}{<}stdbool.\+h\texorpdfstring{$>$}{>}).

Ejemplo usando una variable booleana\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{bool} bandera = \textcolor{keyword}{true};}

\DoxyCodeLine{    puts(\textcolor{stringliteral}{"{}La bandera está activa"{}});}
\DoxyCodeLine{\}}

\end{DoxyCode}


(en C, cualquier expresión entera distinta de 0 se trata como verdadera).

\item switch\+: selecciona entre varias ramas según el valor de una expresión integral (int, char, enum, etc.).
\end{DoxyItemize}\hypertarget{group__apuntes_autotoc_md6}{}\doxyparagraph{Sintaxis básica\+:}\label{group__apuntes_autotoc_md6}

\DoxyCodeLine{    \textcolor{keywordflow}{case} constante1:}
\DoxyCodeLine{        \textcolor{comment}{/* sentencia(s) */}}
\DoxyCodeLine{        \textcolor{keywordflow}{break};}

\DoxyCodeLine{        \textcolor{comment}{/* alternativa si no coincide ningún case */}}
\DoxyCodeLine{\}}

\end{DoxyCode}

\begin{DoxyItemize}
\item break\+: termina la ejecución del switch para evitar \char`\"{}fall-\/through\char`\"{} (caída a la siguiente etiqueta).
\item Fall-\/through\+: si omites break, la ejecución continúa en el siguiente case — útil para agrupar casos\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{switch} (opcion) \{}
\DoxyCodeLine{    \textcolor{keywordflow}{case} 1:}
\DoxyCodeLine{    \textcolor{keywordflow}{case} 2:}
\DoxyCodeLine{        puts(\textcolor{stringliteral}{"{}Opción 1 o 2"{}});}
\DoxyCodeLine{        \textcolor{keywordflow}{break};}
\DoxyCodeLine{    \textcolor{keywordflow}{default}:}
\DoxyCodeLine{        puts(\textcolor{stringliteral}{"{}Otra opción"{}});}
\DoxyCodeLine{\}}

\end{DoxyCode}

\item Notas rápidas\+:
\begin{DoxyItemize}
\item Las etiquetas case deben ser constantes en tiempo de compilación.
\item default es opcional y se ejecuta si ningún case coincide.
\item El orden de los case no afecta la comparación (se compara el valor). 
\end{DoxyItemize}
